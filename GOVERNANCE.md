# Governance

`wickra-embed` is maintained under a BDFL (benevolent dictator for life) model,
consistent with the rest of the Wickra ecosystem.

## Roles

- **Maintainer** — reviews and merges changes, sets direction, cuts releases.
  The current maintainers are listed in [MAINTAINERS.md](MAINTAINERS.md).
- **Contributor** — anyone who opens an issue or a pull request.

## Decision making

Day-to-day changes are merged by a maintainer once CI is green and the change
has been reviewed. Larger or breaking changes (the `Indicator` contract, the
no-alloc C ABI, the set of supported targets, or the byte-parity guarantee) are
discussed in an issue first and decided by maintainer consensus; the lead
maintainer breaks ties.

## Changing this document

Governance changes are proposed by pull request and accepted at the maintainers'
discretion.
