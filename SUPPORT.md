# Support

Thanks for using `wickra-embed`.

## Questions and usage help

For "how do I…" questions, open a
[GitHub Discussion](https://github.com/wickra-lib/wickra-embed/discussions).

## Bugs and feature requests

Open a [GitHub issue](https://github.com/wickra-lib/wickra-embed/issues) using
the bug-report or feature-request template. For a bug, please include the
version, the Rust toolchain and target (host / `thumbv7em-none-eabihf` /
`thumbv6m-none-eabi`), the indicator and parameters, the input values, and the
expected vs actual output — ideally the `wickra-core` value it should match.

## Security

Do not open a public issue for a vulnerability. See [SECURITY.md](SECURITY.md)
for private reporting.
